## Installation
Clone the repository:
```bash
git clone https://github.com/stt08/car-assembly.git
```
Install the dependencies (server):
```bash
# path: ./car-assembly
npm install
```
Install the dependencies (client):
```bash
# path: ./car-assembly/client
npm install
```

## Run the project
Server side
```bash
# path: ./car-assembly
npm run dev
```
Client side
```bash
# path: ./car-assembly/client
npm run serve
```
